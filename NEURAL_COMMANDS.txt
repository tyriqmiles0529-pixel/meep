# Neural Network Training - Quick Command Reference

## Installation (Required)
pip install torch pytorch-tabnet

# Verify installation
python -c "import torch; import pytorch_tabnet; print('Ready!')"

## Training Commands (Neural network is now DEFAULT)

# Standard training (uses neural network automatically)
python train_auto.py --verbose --fresh

# Force CPU device
python train_auto.py --verbose --fresh --neural-device cpu

# Force GPU device (requires CUDA)
python train_auto.py --verbose --fresh --neural-device gpu

# Quick test (20 epochs, recent data only)
python train_auto.py --neural-epochs 20 --player-season-cutoff 2020

# Full training with all options
python train_auto.py ^
    --verbose ^
    --fresh ^
    --neural-epochs 100 ^
    --neural-device auto ^
    --enable-window-ensemble ^
    --lgb-log-period 50

# Disable neural network (NOT recommended, for testing only)
python train_auto.py --verbose --fresh --disable-neural

## Daily Workflow (No Changes)
python riq_analyzer.py    # Get predictions
python evaluate.py        # Evaluate results

## Troubleshooting

# Check if GPU available
python -c "import torch; print(f'CUDA: {torch.cuda.is_available()}')"

# Test import
python -c "from neural_hybrid import NeuralHybridPredictor; print('OK')"

# View training metadata
type models\training_metadata.json

## Training Time Estimates

CPU (Intel i7 / AMD Ryzen 7):
- Standard:      30-45 minutes
- Neural hybrid: 40-60 minutes

GPU (NVIDIA RTX 3060+):
- Standard:      30-45 minutes
- Neural hybrid: 10-15 minutes (faster!)

## Expected Accuracy Gains

Points:   +2-6% (most significant)
Rebounds: +0-5%
Assists:  +0-6%
Threes:   +0-8% (most significant)

## Storage Requirements

Standard:      ~10-20 MB (models folder)
Neural hybrid: ~100 MB (includes TabNet weights)

## Neural Network is Now Default

✅ Neural hybrid is ALWAYS used (unless explicitly disabled)

Only disable with --disable-neural if:
- Quick testing/debugging
- Disk space severely limited (<100 MB)
- Comparing against baseline LightGBM

⚠️ NOT recommended for production use
